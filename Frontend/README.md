This is the frontend of the Uber Clone, built with modern React features and high-performance animations. It provides a seamless user experience for both riders and captains.

---

## 🔥 Features

- 🏎️ **Optimized with Vite**: Lightning-fast development and build times.
- ✨ **Fluid Animations**: Implemented with GSAP for a premium mobile-app-like feel.
- 🎨 **Modern Styling**: Tailwind CSS v4 for a highly maintainable and responsive UI.
- 🛣️ **Seamless Routing**: Integrated with React Router Dom for smooth page transitions.
- 🔒 **Authenticated Views**: Protected routes for both users and captains using React Context.
- 🗺️ **Interactive UI**: Custom-built search panels and bottom sheets for ride booking.

---

## 🛠️ Stack

- **Framework**: React 19
- **Build System**: Vite 7
- **Styling**: Tailwind CSS 4
- **Animation**: GSAP + @gsap/react
- **Network**: Axios
- **Icons**: Remix Icon & Lucide React
- **Notifications**: (Optional, add your toast library if any)

## 🚀 Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Setup Environment**
   Create a `.env` file:
   ```env
   VITE_API_URL=http://localhost:5000
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

## 📸 Component Highlights

- **Auth Screens**: clean, minimal designs for Login and Signup.
- **Home Dashboard**: Modern map-based UI (via conditional rendering/overlays).
- **Search Panel**: Interactive bottom-sheet style search for locations.
- **Vehicle Selection**: Visual selection for car, bike, or auto.
